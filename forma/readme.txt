Для передачи формы я использовал метод POST, ибо GET не безопасен.
Для той же безопасности я отфильтровал полученные данные - использовал функцию для удаления лишних пробелов и функцию для фильтрации тегов
Создал переменную с необходимыми параметрами для дальнейшей фильтрации возраста.
С помощью условий проверил необходимые требования к полученным переменным.
Первое услови проверяет переданное поле "имя". !isset для проверки пустое поле поле передалось или нет. 
mb_strlen считает кол-во символов в переданном поле и далее сравнивается, чтобы количество символов не привышало значение 30. 
С помощью регулярного выражения проверил, чтобы передавались только русские и английские символы.
Второе условие проверяет переданное поле "номер телефона". !isset для проверки пустое поле поле передалось или нет. 
С помощью регулярного выражения проверил, чтобы передавались только необходимые символы и ограничил длину передаваемого номера.
Второе условие проверяет переданное поле "возраст". !isset для проверки пустое поле поле передалось или нет.
С помощью функции filter_var с параметром FILTER_VALIDATE_INT проверил, чтобы передавалось целое число с определенным диапозоном.
